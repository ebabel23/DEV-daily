List of Piano-related Variables via tp-push

How To use the vars?

In Plain HTML: Put Name in double { { custom.variable_goes_here } }

In Custom-JS in a Template: scope.[object]

Outside Piano in JS (eg. in Flows with RunJS): tp.customVariables.VARNAME

  • e2p_refUrl
  • Usage: Shows-Referring Page
  • Type: String / URL
  • Example: {{ custom.e2p_refUrl }}
  • e2p_promoSlot
  • Usage: Used to control Gutschein-Slot
  • Type: Boolean / from section-property
  • Example: {{ custom.e2p_promoSlot }}
  • e2p_titleCode
  • Usage: Used identify title
  • Type: String / e.g. nrw-wp
  • Example: {{ custom.e2p_titleCode }}
  • e2p_isSection
  • Usage: Used identify if page is a section
  • Type: Boolean
  • Example: {{ custom.e2p_isSection }}
  • e2p_isArticle
  • Usage: Used identify if page is an article
  • Type: Boolean
  • Example: {{ custom.e2p_isArticle }}
  • e2p_articleType
  • Usage: Shows escbnic-article-type of page
  • Type: String / eg. Nachricht
  • Example: {{ custom.e2p_articleType }}
  • e2p_articleId
  • Usage: Shows escenic-article-id
  • Type: String (Number?)
  • Example: {{ custom.e2p_articleId }}
  • e2p_paidState
  • Usage: Shows computed premium-state of page
  • Type: Boolean
  • Example: {{ custom.e2p_paidState }}
  • e2p_isPlusPage
  • Usage: Shows if page is a sales-page
  • Type: Boolean from section-parameter
  • Example: {{ custom.e2p_isPlusPage }}
  • e2p_consentLevel
  • Usage: Shows if active user gave consent
  • Type: Boolean from section-parameter
  • Example: {{ custom.e2p_consentLevel }}
